# Second Activity

De second activity app kwam voort uit een oefening, waarbij ik wilde proberen een nieuw scherm te openen.
Dit omdat dit enorm belangrijke stappen zijn in het maken van een app. Deze app heb ik dus gemaakt zonder dat dit een opdracht was van de Google Course.
De reden dat ik deze opdracht ben begonnen is voornamelijk omdat de course voor mij te lang duurde en te veel dubbele inhoud had.
Het doel was om meer te leren te krijgen over hoe dat je apps bouwt in Android Studio. Ik wil de basis kennen, 
zodat ik een goed oordeel kan geven over de leercurve van Kotlin in combinatie met Android Studio.

---

## 🚀 Features
- Een tweede pagina openen met behulp van een knop.
  
---
